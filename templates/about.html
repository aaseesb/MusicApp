Project Description

This is a web application that allows users to search for songs by their titles. It fetches song data using the ITunes and Deezer API's and displays information about the song including the title, singer, album, genre and publication year. Additionally, it allows you to play the song. The project is built using Flask, HTML, CSS, JavaScript and Python.
